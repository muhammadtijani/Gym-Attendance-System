
#  Gym Attendance & Management System
A lightweight CRUD (Create, Read, Update, Delete) application built to demonstrate full-stack data engineering principles.

# Features
Member Management: Add new members to a persistent SQL database.
Relational Tracking: Linked "Attendance" table to track member check-ins via Foreign Keys.
Real-time Dashboard: View the latest 5 check-ins instantly.

# Tech Stack
Backend: Python (Flask)
Database: SQL (SQLite) / SQLAlchemy ORM
Frontend: HTML5 / CSS3

# Why this matters for Data Analysis
This project demonstrates the ability to build data collection pipelines. It shows an understanding of Relational Database Design (One-to-Many relationships) and how raw user interactions are transformed into structured data for analysis.
